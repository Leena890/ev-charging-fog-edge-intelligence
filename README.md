# Resilient Edge-to-Cloud EV Charging Intelligence Platform

An end-to-end IoT platform designed to optimize data bandwidth and provide zero-latency safety interventions for modern Electric Vehicle (EV) charging networks.

## 🚀 Key Features
* **Edge ML Isolation Forest:** Localized anomaly detection to trigger immediate physical emergency power shutdowns.
* **Deadband Data Filtering:** Dropping high-frequency micro-fluctuations at the edge to optimize network bandwidth.
* **Decoupled Serverless Cloud Pipeline:** Robust asynchronous data ingestion using AWS SQS, Lambda, and DynamoDB.
* **Operator Analytics Dashboard:** Live monitoring interface built with Streamlit.
